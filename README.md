# Portfolio — AETHER

**Personal landing page** · Security Researcher · Bug Bounty Hunter

A single-page portfolio link-in-bio site built with raw HTML + CSS. No frameworks, no bloat — just clean design, dark aesthetics, and a terminal-inspired vibe.

## 🚀 Live

**[→ aether-dz.github.io/portfolio](https://aether-dz.github.io/portfolio/)**

## ✨ Features

- **Dark cyber aesthetic** — animated grid background, ambient glow orbs, conic-gradient profile ring
- **Typing animation** — multi-title cycling typewriter effect (Security Researcher → Bug Bounty Hunter → etc.)
- **Arsenal showcase** — 16 skill tags with hover glow
- **Social hub** — GitHub, LinkedIn, Instagram, X, Telegram — all in one place
- **Stats panel** — Reports, Platforms, Curiosity metrics
- **Project cards** — linked showcase with hover slide
- **Fully responsive** — mobile-first, adaptive layout
- **Zero dependencies** — pure HTML + CSS + vanilla JS

## 🧰 Stack

| Layer | What |
|-------|------|
| Markup | HTML5 |
| Styling | CSS3 (custom properties, animations, grid) |
| Icons | Font Awesome 6 (CDN) |
| Fonts | JetBrains Mono + Inter (Google Fonts) |
| Hosting | GitHub Pages |

## 🏗️ Local Dev

```bash
# Clone
git clone https://github.com/Aether-DZ/portfolio.git
cd portfolio

# Serve locally
python3 -m http.server 8000
# → http://localhost:8000
```
